Two Buttons + LED Logic

TIER 3 - MEDIUM PRIORITY

Function: LED responds to button combinations using AND/OR logic. Tests boolean operators and multiple inputs.

Connections

  • Button 1: One terminal → 5V, Other → Pin 2 + 10kΩ to GND
  • Button 2: One terminal → 5V, Other → Pin 3 + 10kΩ to GND
  • LED: Anode → 220Ω → Pin 13, Cathode → GND

Complete Arduino Code - AND Logic

// Two Buttons AND Logic
// LED ON only when BOTH buttons pressed

const int button1Pin = 2;
const int button2Pin = 3;
const int ledPin = 13;

void setup() {
  pinMode(button1Pin, INPUT);
  pinMode(button2Pin, INPUT);
  pinMode(ledPin, OUTPUT);
}

void loop() {
  bool button1 = digitalRead(button1Pin);
  bool button2 = digitalRead(button2Pin);

  // AND logic: Both must be HIGH
  if (button1 == HIGH && button2 == HIGH) {
    digitalWrite(ledPin, HIGH);
  } else {
    digitalWrite(ledPin, LOW);
  }

  delay(10);
}

Alternative: OR Logic

// OR Logic: LED ON when EITHER button pressed
if (button1 == HIGH || button2 == HIGH) {
  digitalWrite(ledPin, HIGH);
} else {
  digitalWrite(ledPin, LOW);
}

Alternative: XOR Logic

// XOR Logic: LED ON when ONLY ONE button pressed (not both, not neither)
if ((button1 == HIGH && button2 == LOW) || (button1 == LOW && button2 == HIGH)) {
  digitalWrite(ledPin, HIGH);
} else {
  digitalWrite(ledPin, LOW);
}
