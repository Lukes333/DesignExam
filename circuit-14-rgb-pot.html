RGB LED + Potentiometer

TIER 3 - MEDIUM PRIORITY

Function: Turn potentiometer to change RGB LED color through spectrum. Tests multiple PWM outputs and color mixing.

Components & Connections (Common Cathode RGB LED)

  • RGB LED Cathode (longest pin): GND
  • Red Anode: 220Ω → Pin 9 (PWM)
  • Green Anode: 220Ω → Pin 10 (PWM)
  • Blue Anode: 220Ω → Pin 11 (PWM)
  • Potentiometer: Left → 5V, Middle → A0, Right → GND
Note: All LED pins must be PWM (3, 5, 6, 9, 10, 11) for color mixing!

Complete Arduino Code

// RGB LED Color Control

const int potPin = A0;
const int redPin = 9;
const int greenPin = 10;
const int bluePin = 11;

int potValue = 0;

void setup() {
  pinMode(redPin, OUTPUT);
  pinMode(greenPin, OUTPUT);
  pinMode(bluePin, OUTPUT);
}

void loop() {
  potValue = analogRead(potPin);  // 0-1023

  // Map pot to color spectrum (0-765)
  // 0-255: Red to Yellow
  // 256-511: Yellow to Green
  // 512-767: Green to Cyan
  // 768-1023: Cyan to Blue

  int colorValue = map(potValue, 0, 1023, 0, 765);

  int red, green, blue;

  if (colorValue < 256) {
    // Red to Yellow (increase green)
    red = 255;
    green = colorValue;
    blue = 0;
  } else if (colorValue < 512) {
    // Yellow to Green (decrease red)
    red = 255 - (colorValue - 256);
    green = 255;
    blue = 0;
  } else {
    // Green to Blue (increase blue, decrease green)
    red = 0;
    green = 255 - (colorValue - 512);
    blue = (colorValue - 512);
  }

  // Set RGB LED
  analogWrite(redPin, red);
  analogWrite(greenPin, green);
  analogWrite(bluePin, blue);

  delay(10);
}

// Pot left: Red
// Pot middle: Yellow/Green
// Pot right: Blue

Simple Version (Fixed Colors)

// Simpler: Switch between 5 fixed colors

void loop() {
  potValue = analogRead(potPin);

  if (potValue < 205) {
    // RED
    analogWrite(redPin, 255);
    analogWrite(greenPin, 0);
    analogWrite(bluePin, 0);
  } else if (potValue < 410) {
    // YELLOW
    analogWrite(redPin, 255);
    analogWrite(greenPin, 255);
    analogWrite(bluePin, 0);
  } else if (potValue < 615) {
    // GREEN
    analogWrite(redPin, 0);
    analogWrite(greenPin, 255);
    analogWrite(bluePin, 0);
  } else if (potValue < 820) {
    // CYAN
    analogWrite(redPin, 0);
    analogWrite(greenPin, 255);
    analogWrite(bluePin, 255);
  } else {
    // BLUE
    analogWrite(redPin, 0);
    analogWrite(greenPin, 0);
    analogWrite(bluePin, 255);
  }

  delay(10);
}

Testing Checklist

  1. ✅ All 3 LED pins are PWM
  2. ✅ Each LED pin has 220Ω resistor
  3. ✅ Common cathode to GND (or common anode to 5V)
  4. ✅ Pot turns → LED color changes
  5. ✅ Can produce red, green, blue, yellow, cyan
