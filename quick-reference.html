Essential Arduino Functions

Digital I/O

pinMode(pin, OUTPUT); // Set as output

digitalWrite(pin, LOW); // Set LOW (0V)

Servo Library Reference

Servo Library Functions

1. Include library (top of code):
#include <Servo.h>
2. Create servo object (before setup):
Servo myServo; // Can name anything
3. Attach to pin (in setup):
myServo.attach(9); // Pin must be PWM
4. Set position (in loop):
myServo.write(90); // 0-180 degrees
5. Read position:
int pos = myServo.read(); // Returns angle
6. Detach servo (optional):
myServo.detach(); // Stops PWM signal
Common Servo Mistakes:
  • ‚ùå Forgetting #include <Servo.h> at top
  • ‚ùå Not calling .attach() in setup()
  • ‚ùå Calling .attach() in loop() (should be setup only)
  • ‚ùå Using analogWrite() instead of .write()
  • ‚ùå Servo on non-PWM pin

Component Specifications

HC-SR04 Ultrasonic

  • Voltage: 5V DC
  • Range: 2-400 cm
  • Accuracy: ¬±3mm
  • Trigger: 10Œºs HIGH pulse
  • Echo: Duration proportional to distance
  • Angle: 15¬∞ detection cone

Servo Motor

  • Voltage: 4.8-6V (usually 5V)
  • Range: 0-180 degrees
  • Signal: PWM 50Hz, 1-2ms pulse
  • Wires: Red=5V, Brown=GND, Orange=Signal
  • Library: Required (#include <Servo.h>)

TMP36 Temperature

  • Voltage: 2.7-5.5V
  • Range: -40¬∞C to +125¬∞C
  • Accuracy: ¬±2¬∞C
  • Output: 0.5V at 0¬∞C
  • Scale: +10mV per ¬∞C
  • Pins: Left=5V, Middle=Signal, Right=GND

L293D Motor Driver

  • Pin 1 (Enable): PWM for speed
  • Pin 2,7 (Inputs): Direction control
  • Pin 3,6 (Outputs): To motor
  • Pin 4,5 (GND): Ground (both!)
  • Pin 8 (VCC2): Motor power (9V)
  • Pin 16 (VCC1): Logic power (5V)

Quick Troubleshooting Guide

Code Won't Compile

  • ‚úì Missing semicolon ; at end of line
  • ‚úì Mismatched { } braces
  • ‚úì Forgot #include for library
  • ‚úì Variable not declared (const int missing)
  • ‚úì Wrong capitalization (pinMode not pinmode)

LED Not Working

  • ‚úì LED backwards (check anode/cathode)
  • ‚úì No resistor (LED burned out)
  • ‚úì Wrong pin number in code
  • ‚úì pinMode not set to OUTPUT
  • ‚úì digitalWrite LOW when should be HIGH

Sensor Reading Wrong

  • ‚úì Wrong pin in analogRead()
  • ‚úì No voltage divider (LDR, FSR)
  • ‚úì Wrong formula or conversion
  • ‚úì Integer division (use 5.0 not 5)
  • ‚úì Sensor backwards (check pinout)

Servo Not Moving

  • ‚úì Forgot #include <Servo.h>
  • ‚úì Didn't call .attach() in setup()
  • ‚úì Using analogWrite() instead of .write()
  • ‚úì Servo on non-PWM pin
  • ‚úì Servo wires swapped

Serial Monitor Issues

  • ‚úì Forgot Serial.begin(9600) in setup()
  • ‚úì Wrong baud rate selected in monitor
  • ‚úì Using pins 0/1 (RX/TX) for other purposes
  • ‚úì No Serial.print() statements in code

Button Not Responding

  • ‚úì No pull-down/pull-up resistor
  • ‚úì pinMode set to OUTPUT (should be INPUT)
  • ‚úì No debounce delay
  • ‚úì Wrong logic (checking LOW when should be HIGH)
